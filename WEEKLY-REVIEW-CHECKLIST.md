# 週次レビュー クイックチェックリスト

**使い方:** 毎週、Claude Codeに「週次レビュー」と言うだけで、このチェックリストに沿ってレビューが開始されます。

---

## 🚀 クイックスタート

### ユーザーが言うこと:
```
「週次レビュー」
```

### AIが自動的に実行すること:
1. ✅ 現在のMonth（フェーズ）を確認
2. ✅ GA4データの提供を依頼
3. ✅ 目標KPIと実績を比較
4. ✅ 達成・未達成の分析
5. ✅ 次週のアクション提案

---

## 📊 必要なGA4データ（ユーザーが準備するもの）

### 最低限必要なデータ（3つ）:
```
□ 過去7日間の訪問者数（Users）
□ 過去7日間のnankan-analyticsクリック数
□ 過去7日間のオーガニック検索の割合
```

### 推奨データ（可能なら追加）:
```
□ ページビュー数
□ 平均セッション時間
□ 直帰率
□ 人気ページTOP5
```

### GA4での確認方法:

**1. 訪問者数:**
```
GA4 > レポート > ライフサイクル > エンゲージメント > 概要
期間: 過去7日間
指標: ユーザー
```

**2. nankan-analyticsクリック数:**
```
GA4 > レポート > ライフサイクル > エンゲージメント > イベント
イベント名: click
パラメータでフィルタ: link_url contains "nankan-analytics.com"
期間: 過去7日間
```

**3. オーガニック検索の割合:**
```
GA4 > レポート > ライフサイクル > 集客 > トラフィック獲得
期間: 過去7日間
セッションのデフォルト チャネル グループで「Organic Search」を確認
```

## 🔄 レビューの流れ（例）

### 週次レビューの例:

**ユーザー:**
```
週次レビュー
```

**AI:**
```
✅ 週次レビューを開始します。

現在は Month 1（基盤構築フェーズ）の第2週目ですね。

GA4で以下のデータを確認していただけますか？
1. 過去7日間の訪問者数（Users）
2. nankan-analyticsへのクリック数
3. オーガニック検索の割合

データを教えていただければ、目標との比較と次週のアクションを提案します。
```

**ユーザー:**
```
訪問者: 25人
クリック: 3回
オーガニック: 8%
```

**AI:**
```
📊 Week 2 レビュー結果

【実績 vs 目標】
- 訪問者: 25人（目標: 50-100人の中央値75人）
  達成率: 33% 🔴未達成

- nankan-analyticsクリック: 3回（目標: 2-5回）
  達成率: 60% 🟡ほぼ達成

- オーガニック流入: 8%（目標: 5-10%）
  達成率: 80% 🟡ほぼ達成

【分析】
✅ オーガニック流入とクリック数は順調
🔴 訪問者数が目標の1/3、要改善

【次週のアクション（優先度順）】
1. Search Consoleでインデックス状況確認（未登録ページがある可能性）
2. SNS（X/Twitter）で南関競馬関連投稿、サイトへの誘導
3. nankan-analyticsからの逆リンク強化を依頼
4. 内部リンク構造の見直し（クローラビリティ向上）
5. Google Search Consoleにサイトマップ再送信

実施しますか？
```

## ⚠️ 重要な注意事項

### AIへの指示（毎回実行すること）:
1. トリガーキーワードを見たら、CLAUDE.mdの「定期的な任務」セクションを読む
2. GA4-GROWTH-ROADMAP.mdを開いて、現在のMonthを確認
3. 手順に従って、データ要求 → 分析 → 提案を実施
4. 忘れない: **これは任務である**

### ユーザーへの推奨:
- 週次レビュー: 毎週同じ曜日・時間に実施（習慣化）
- 月次レビュー: 毎月1日に実施
- GA4データはスクリーンショットで共有すると正確
