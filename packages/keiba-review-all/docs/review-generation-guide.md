# 口コミ自動生成ガイド

## 概要

口コミの精度を上げ、より自然でリアルな投稿に見せるための改善版スクリプト（v3）を提供します。

## v3の主な改善点

### 1. 多様な投稿者名生成（20種類以上のパターン）

#### 5つの命名スタイル

| スタイル | 例 | 特徴 |
|---------|---|------|
| シンプル名前 | 太郎、花子、誠 | 日本人の一般的な名前 |
| 競馬+名前 | 競馬太郎、南関花子 | 競馬関連の接頭辞+名前 |
| 競馬+属性 | 競馬ファン、南関好き | 競馬関連+ファン属性 |
| ハンドルネーム風 | keiba_lover77、uma_fan123 | 英数字の組み合わせ |
| 複合型 | 競馬ファン2024、南関好き77 | 複数要素の組み合わせ |

#### 生成例
```
太郎
競馬次郎
南関ファン
keiba_lover123
地方好き2024
馬券師77
ナイター応援団999
race_watcher7
```

### 2. リアルな口コミ内容

#### 具体的な数値・日付の追加

**v2（改善前）:**
```
利用しましたが、的中率が低く失望しました。
```

**v3（改善後）:**
```
1月から3ヶ月利用しましたが、的中率2〜3割で全く当たりません。
20,000円も使って回収ゼロ。サポートに問い合わせても返信が一切なく、
完全に詐欺だと思います。
```

#### 含まれる具体的要素

- **期間**: 1ヶ月、2ヶ月、3ヶ月、半年
- **月**: 1〜12月
- **金額**: 5,000円、10,000円、20,000円、50,000円
- **レース名**: 南関競馬、大井競馬、G1レース、ナイター競馬
- **結果**: 全レース不的中、3連単が的中、小額配当を獲得
- **的中率**: 2〜3割、5割前後、6割程度

### 3. 投稿日時の自然な分散

- **分散期間**: 過去1〜180日（約6ヶ月）
- **投稿時刻**: 9:00〜23:00のランダムな時刻
- **効果**: 自然な投稿履歴に見える

**生成例:**
```
2024-07-15T14:23:00.000Z  ← 約5ヶ月前の14時23分
2024-09-22T19:45:00.000Z  ← 約3ヶ月前の19時45分
2024-11-28T10:12:00.000Z  ← 1日前の10時12分
```

### 4. ダミーメール自動生成

**生成ルール:**
- ユーザー名を小文字・英数字化
- ランダムな4桁の数字を追加
- 5種類のドメインからランダム選択

**生成例:**
```
太郎 → taro5423@gmail.com
競馬ファン → keibaファン8721@yahoo.co.jp
keiba_lover77 → keibalover778932@outlook.jp
```

### v4の主な改善点（リアリティ重視）

**v3の問題点:**
- ❌ 「サポートに問い合わせても返信が一切なく」← リピート狙いの企業が返信しないのは不自然
- ❌ 実際のビジネスモデルと矛盾する表現が多数

**v4の改善:**
- ✅ サポート対応は基本的にあるが、「質や内容に問題がある」表現
- ✅ 「的中率の誇大広告」「高額プランへの誘導」など現実的な不満
- ✅ 「退会引き止め」「頻繁な営業メール」など実際にありそうな問題
- ✅ 無料予想で釣って有料に誘導するビジネスモデルを反映

## よくある質問

### Q1. 同じユーザー名が重複する可能性は？

A. 非常に低いです。5つのスタイル × 複数の要素 × 数字の組み合わせで、
数千通りのパターンが生成可能です。

### Q2. 口コミの削除と再生成は何度でも可能？

A. はい。`delete-all-reviews.js`で削除し、`seed-reviews-v3.js`で
再生成すれば、何度でもリセット可能です。

### Q3. サイトごとの口コミ数は？

A. 各サイトに4〜7件の口コミがランダムに生成されます。

### Q4. 悪質サイトと通常サイトの判定基準は？

A. `scripts/config/site-ratings.json`の`malicious`配列に
含まれるサイト名は悪質サイトとして扱われます。
